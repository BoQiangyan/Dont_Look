# Dont_Look

《你回头干啥》是一款轻量级 H5 / WebView 像素街机风回合制对战小游戏。

游戏围绕“你到底回不回头”的心理博弈展开。玩家需要在正面攻击、背后偷袭、假动作、防守判断之间做选择，通过猜测对方行动来取得胜利。

## 项目简介

本项目是一款网页小游戏，当前支持单人模式与局域网双人对战模式。

当前版本主要功能包括：

- 单人模式
- 局域网双人对战
- 攻防轮换制
- 像素风角色与舞台
- PNG 按钮与效果词
- 基础音效
- WebSocket 双人同步
- 固定世界视角战斗画面

## 游戏角色

当前版本包含两个角色：

| 角色 | 位置 | 说明 |
| --- | --- | --- |
| 马大愣 | 左侧 | 正面硬刚派 |
| 麦克老六 | 右侧 | 背后阴人派 |

在双人模式中，战斗画面采用固定世界视角：

- 左边永远是马大愣
- 右边永远是麦克老六
- 两台电脑看到的是同一套战斗画面
- 马大愣出拳时，两边电脑都显示左侧角色出拳
- 麦克老六出拳时，两边电脑都显示右侧角色出拳

## 核心玩法

游戏采用攻防轮换制。

每个小回合中：

- 一方是进攻方
- 一方是防守方
- 回合结束后交换攻防身份

进攻方需要判断防守方会不会回头。

防守方需要判断进攻方会从正面攻击，还是从背后偷袭。

## 行动说明

### 进攻方行动

| 行动 | 说明 |
| --- | --- |
| 正面来一下 | 正面攻击，克制猛回头 |
| 投掷铁块 | 背后攻击，克制不回头 |
| 啥也不干 | 假动作，骗对方判断并积攒心眼值 |
| 连着来两下 | 心眼值满 3 点后可用，不能被完全防住 |

### 防守方行动

| 行动 | 说明 |
| --- | --- |
| 就不回头 | 防正面攻击 |
| 猛回头 | 防背后铁块 |

## 结算规则

| 进攻方行动 | 防守方就不回头 | 防守方猛回头 |
| --- | --- | --- |
| 正面来一下 | 被防住，0 伤害 | 命中，造成 1 点伤害 |
| 投掷铁块 | 命中，造成 2 点伤害 | 被防住，0 伤害 |
| 啥也不干 | 不造成伤害，进攻方积攒心眼值 | 不造成伤害，进攻方获得更多心眼值 |
| 连着来两下 | 造成 2 点伤害 | 造成 1 点伤害 |

## 数值规则

| 项目 | 数值 |
| --- | --- |
| 初始血量 | 5 |
| 心眼值上限 | 3 |
| 正面一拳伤害 | 1 |
| 投掷铁块伤害 | 2 |
| 连着来两下伤害 | 2 |
| 猛回头防连击后伤害 | 1 |
| 投掷铁块冷却 | 10 秒 |
| 单人每回合选择时间 | 3 秒 |
| 双人每回合选择时间 | 30 秒 |

## 本地运行教程

### 1. 安装 Node.js

运行本项目需要安装 Node.js。

可以前往 Node.js 官网下载安装：

```text
https://nodejs.org/
```

安装完成后，在终端中检查版本：

```bash
node -v
```

如果能看到版本号，说明 Node.js 已安装成功。

### 2. 进入项目目录

打开终端，进入项目目录：

```bash
cd project
```

如果你的项目在 Windows 上，可以类似这样进入：

```powershell
cd "F:\Hackathon\don't_look\project"
```

### 3. 安装依赖

第一次运行前，需要安装依赖：

```bash
npm install
```

### 4. 启动页面服务

运行：

```bash
npm run dev
```

启动成功后，终端会显示本地预览服务地址。

### 5. 启动双人对战服务

如果只玩单人模式，可以只启动页面服务。

如果要玩双人对战，需要再打开一个新的终端，进入同一个项目目录：

```bash
cd project
```

然后运行：

```bash
npm run server
```

双人模式需要同时保持两个终端运行：

| 终端 | 命令 | 作用 |
| --- | --- | --- |
| 终端 1 | `npm run dev` | 启动网页服务 |
| 终端 2 | `npm run server` | 启动双人对战服务器 |

### 6. 局域网双人对战教程

双人对战适合同一 Wi-Fi / 同一局域网下测试。

操作流程：

1. 主机电脑连接 Wi-Fi。
2. 朋友电脑连接同一个 Wi-Fi。
3. 主机电脑启动网页服务：

```bash
npm run dev
```

4. 主机电脑再打开一个终端，启动双人服务器：

```bash
npm run server
```

5. 主机电脑查看自己的局域网 IP。

Windows 可以在 PowerShell 中输入：

```powershell
ipconfig
```

找到类似下面这一行：

```text
IPv4 地址 . . . . . . . . . . . . : 192.168.124.210
```

6. 两台电脑都在浏览器中访问主机电脑地址。

示例：

```text
http://192.168.124.210:5173/
```

7. 两边进入页面后，选择双人对战。
8. 一方选择马大愣，另一方选择麦克老六。
9. 匹配成功后即可开始对战。

### 7. 常见问题

#### 页面按钮没有反应

不要直接双击或右键打开 `index.html`。

错误方式：

```text
file:///.../index.html
```

正确方式是先启动服务：

```bash
npm run dev
```

然后通过浏览器访问服务地址。

#### 双人模式无法连接

请检查：

- 两台电脑是否连接同一个 Wi-Fi
- 主机电脑是否启动了 `npm run server`
- 主机电脑是否启动了 `npm run dev`
- 朋友访问的是否是主机电脑 IP
- Windows 防火墙是否允许 Node.js 访问网络

#### 修改后页面没变化

可能是浏览器缓存。

可以尝试：

- 强制刷新页面
- 关闭标签页重新打开
- 使用无痕窗口
- 确认终端服务已经重启
